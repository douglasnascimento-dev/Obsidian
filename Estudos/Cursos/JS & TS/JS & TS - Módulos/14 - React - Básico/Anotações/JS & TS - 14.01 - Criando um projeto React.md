# 🌳Criando um projeto com React

```powershell
npm create vite@latest
```

O Comando irá abrir o menu com as seguintes perguntas:
1. **Project name?** (Nome do projeto?)
    - Digite o nome da pasta para o seu novo projeto ou . para criar na própria pasta. 
- **Select a framework?** (Selecione um framework?)
    - Use as setas do teclado para navegar e selecionar **React**.
- **Select a variant?** (Selecione uma variante?)
    - Selecione a que preferir.

A Seguir, limpe os arquivos e pastas desnecessários.
