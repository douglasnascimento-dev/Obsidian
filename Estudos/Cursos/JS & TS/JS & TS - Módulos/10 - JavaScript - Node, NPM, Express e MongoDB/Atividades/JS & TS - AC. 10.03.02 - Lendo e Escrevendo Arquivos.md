## 🌳Exercício Avançado: Manipulação de Arquivos com `fs`

### 🌿**Objetivo:** Utilizar o módulo `fs` do Node.js para realizar as seguintes tarefas de leitura e escrita de arquivos:

1. Criar um arquivo chamado `dados.txt` com uma lista de tarefas.
    
2. Adicionar mais informações ao arquivo, incluindo a data atual de modificação.
    
3. Ler o arquivo e exibir a quantidade total de palavras no arquivo.
    
4. Criar um novo arquivo chamado `resumo.txt`, contendo um resumo do arquivo original (somente as 5 primeiras linhas e a quantidade total de palavras).
    
5. Se o arquivo `dados.txt` já existir, adicionar novas tarefas a ele sem sobrescrever as antigas.
