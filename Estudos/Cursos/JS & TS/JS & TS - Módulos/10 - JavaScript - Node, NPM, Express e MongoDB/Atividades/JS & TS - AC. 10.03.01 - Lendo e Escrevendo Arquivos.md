## 🌳 Exercício: Manipulação de Arquivos com `fs`

Crie um script em Node.js que leia um arquivo chamado `dados.json`, altere seu conteúdo e salve as mudanças.

### 🌿 Requisitos:

1. **Leitura do Arquivo**: O script deve verificar se `dados.json` existe. Se existir, deve lê-lo e exibir o conteúdo no console.
2. **Edição do Arquivo**: O script deve adicionar um novo objeto ao array contido no arquivo.
3. **Escrita no Arquivo**: O novo conteúdo deve ser salvo em `dados.json`.
4. **Tratamento de Erros**: O script deve capturar e exibir erros que possam ocorrer durante a leitura ou escrita do arquivo.

#### 🌱 Desafio Bônus:

Em vez de adicionar um objeto fixo, faça com que o usuário possa passar um novo item pelo terminal ao executar o script.
