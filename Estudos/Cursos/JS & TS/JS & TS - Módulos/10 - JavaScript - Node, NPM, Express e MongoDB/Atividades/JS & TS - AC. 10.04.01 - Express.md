## 🌳Exercício: Manipulação de Rotas com Express

Crie um servidor utilizando o Express que tenha as seguintes rotas:
1. **GET /** → Deve retornar uma mensagem de boas-vindas.
2. **GET /sobre** → Deve retornar uma breve descrição do site.
3. **POST /mensagem** → Deve receber uma mensagem do usuário e retornar uma confirmação.
4. **POST /cadastro** → Deve simular o cadastro de um usuário e responder com uma mensagem de sucesso.

### 🌿Bônus

- Modifique a rota **POST /mensagem** para que ela aceite um corpo de requisição no formato JSON e exiba a mensagem enviada no retorno.
