# 🌳Tipos de Referência Vs Tipos Primitivos

🌵**Referência** → Variáveis cujo tipo são classes não devem ser entendidos como caixas, mas sim como ponteiros para um local na memória. 

🌵**Primitivos** → Tipos primitivos são tipos valor, ou seja, seu valor serão caixas e não para ponteiros. 

## 🌿Valor Nulo 

Variáveis com valor 'null' não apontará para nenhum local. 

## 🌿Valores Padrão

Sempre que alocamos qualquer tipo estruturado são atribuídos valores padrão aos seus elementos

> Exemplo: 

> Números > 0
> Boolean > false
> Char > UniCode de 0
> Objeto > Null
