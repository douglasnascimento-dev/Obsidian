Um exemplo de boa aplicação da psicologia das cores é o do [Vinho Cálem](https://calem.pt/) - #Projeto. É possível encontrar mais material sobre o uso da psicologia das cores no Capítulo 13 - Cores.pdf]do Professor Gustavo Guanabara (Curso em Vídeo)

#### 🔹Gradiente em CSS

Para definir um degradê em CSS 3, usamos a propriedade 'background' seguido de 'linear-gradient(*direction*, *color 01*, *color 02*)'. É possível adicionar infinitas cores.

```CSS
background: linear-gradient(to top, #FF7840, #78FA10)
```

A direção pode ser tanto nas direções (to top, to bottom, to right, to left) ou em graus (90 deg, 45 deg)

#### 🔹Shorthand para Fontes

Deve ocorrer da seguinte maineira: 'font style', 'font weight', 'font size' e 'font family'. Como no exemplo abaixo:

```CSS
font: normal 700 3em 'Outfit'
```

Se acaso não for preciso declarar uma determinada função, isto pode ser feito, suprimindo a declaração, mas sempre mantendo a ordem

#### 🔹Font-face em CSS

Para um melhor uso de fontes externas carregadas a partir de downloads é recomendado a criação de fonte-face, da seguinte maineira:

```CSS
@font-face {
	font-family: 'love';
	src: url('caminhoArquivo.otf') format('opentype'), url('caminhoArquivo.ttf') format('truetype'),;
	font-weight: normal;
	font-style: normal;
} /* Declaração: Font-family: 'Love' */
```

Tipos de Formato: 
1. Opentype - (.OTF);
2. Truetype - (.TTF);
3. Embedded-Opentype;
4. Truetype-aat;

#### 🔹Grouping Tags

As 'Grouping Tags' são as tags utilizadas para agrupamento em HTML, surgiram no HTML 5 para substituir o uso exclusivo da div para tal função

| Tag     | Função em HTML                   |
| ------- | :------------------------------- |
| header  | criação de cabeçalhos            |
| main    | criação de conteúdos principais  |
| footer  | criação de rodapés               |
| nav     | criação de menus                 |
| section | criação de uma seção             |
| article | criação de um artigo             |
| aside   | criação de conteúdos periféricos |
