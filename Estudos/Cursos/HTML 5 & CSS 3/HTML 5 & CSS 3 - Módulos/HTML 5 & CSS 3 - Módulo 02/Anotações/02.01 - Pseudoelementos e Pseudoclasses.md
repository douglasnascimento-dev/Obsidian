> [!banner-icon] 🌿PseudoElementos
## 🌳Pseudoelementos

Se tratam de elementos 'falsos', adicionados antes ou depois de elemento CSS. Sua formatação ocorre com 2 dois pontos ( :: ) após o elemento modificado.

```CSS
div:after {
	content: `🌿`
}
```

### 🌿Lista dos principais pseudoelementos

- After: Adiciona elementos após o conteúdo;
- Before: Adiciona elementos antes do conteúdo;

### 🌿Propriedade 'content'


Adiciona um conteúdo nos pseudoelementos, pode ser tanto em formato de texto ou em formato de emoji e ícones


```CSS
content: "Um texto ou um emoji (🌿🌿🌿)"
```

## 🌳Pseudoclasses em CSS

Se relaciona ao estado de um elemento em CSS. É definido com dois pontos ( : ) junto ao elemento utilizado. Como no exemplo:

- Hover: Se o mouse estiver em cima
- Visited: Se o link já foi utilizado
- Ativo: No momento que foi clicado


```CSS
div:hover {
	color: "black"
	background-color: "white"
}
```
