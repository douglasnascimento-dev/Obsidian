# :LiAlbum:  Conceitos

- **Design**: Design em IHC refere-se ao processo de criação de uma solução considerando aspectos como usabilidade, acessibilidade, estética, e como a interface facilita a interação entre o usuário e o sistema.
- **Interface**: Interface é o meio através do qual a interação acontece. Em IHC, a interface é composta pelos elementos visuais ou até outros modos de interação.
- **Affordance**: termo inglês que significa, atualmente, em português "pregnância" que é a qualidade de um objeto que permite ao indivíduo identificar sua funcionalidade sem a necessidade de prévia explicação, o que ocorre intuitivamente ou baseado em experiências anteriores.
- **Interação**: processo através do qual o usuário formula uma intenção, planeja suas ações, atua sobre a interface, percebe e interpreta a resposta do sistema e avalia se seu objetivo foi alcançado.

## :LiServer: Qualidade em IHC

### :LiTabs: Usabilidade

Para Nielsen, a Usabilidade está associada a 5 atributos:

- **Facilidade de Lembrar** - Se refere como a interface ajuda os usuários a lembrarem como realizarem as tarefas.
- **Eficiência** - Se refere a quanto um produto auxilia os usuários na realização de suas tarefas.
- **Facilidade de Aprender** - Se refere ao tempo e esforço necessário para aprender a usar.
- **Segurança no Uso** - Se refere como a interface prove segurança para o usuário explorar a interface e quais medidas o usuário pode tomar para corrigir os erros facilmente.
- **Satisfação do Usuário** - Avaliação subjetiva que expressa o efeito do uso no usuário.

#### :LiNotebook: Experiência do Usuário

São as percepções e respostas de uma pessoa que resultam do uso e/ou uso antecipado de um produto. Incluí as emoções, crenças, preferências, respostas, comportamentos e as realizações do usuário.

### :LiTabs: Acessibilidade

Busca oferecer meios para que o usuário acesse o sistema e interaja com ele, sem que a interface imponha barreiras. 

### :LiTabs: Comunicabilidade

É meio como a interface deve comunicar ao usuário a lógica do design.
