# :LiAlbum:  Abordagens Teóricas

## :LiServer: CRAP
- **Contraste** - Dois elementos são contrastantes entre si, se são totalmente diferentes entre si.
- **Repetição** - Aspectos dos design devem repetir-se no material inteiro.
- **Alinhamento** - Nada deve ser colocado arbitrariamente em uma página.
- **Proximidade** - Itens relacionados entre si devem ser agrupados e aproximados um dos outros

## :LiServer: Engenharia Cognitiva

Desenvolvida por Normam, foca na necessidade de que haver um estágio de interpretação que relaciona as variáveis físicas e psicológicas. Para Normam, a Interação é a travessia entre dois golfos:
- **da Execução** - Quando os usuários entendem como uma interface opera
- **da Avaliação** - Quando os usuários tentam entender o que aconteceu

Os 7 estágios do ciclo da ação são:
1. Objetivo;
2. Planejar;
3. Especificar;
4. Realizar;
5. Perceber;
6. Interpretar; e
7. Avaliar
