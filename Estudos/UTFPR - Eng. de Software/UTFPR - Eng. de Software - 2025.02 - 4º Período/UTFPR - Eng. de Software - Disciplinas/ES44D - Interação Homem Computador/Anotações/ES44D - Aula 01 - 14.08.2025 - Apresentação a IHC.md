# :LiAlbum: O Que é IHC?

É uma área multidisciplinar com foco no design da tecnologia da computação e, em particular, na interação entre humanos e computadores. 

Tem como áreas transversais a Psicologia (auxilia a compreender o comportamento humano), a Ergonomia (Desenvolve e aplica técnicas para a adaptação do seres humanos) e a Antropologia (Se dedica ao estudo aprofundado do ser humano)

## :LiServer: UX vs UI

- UX - User Experience | Design de Interação - Wireframes e Protótipos
- UI - User Interface | Designer Gráfico - Layouts - Cores
