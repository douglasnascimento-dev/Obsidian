## 🌿Dados Abertos

> São todo conteúdo que se pode utilizar de forma livre, reutiliza-las ou redistribui-las, sem restrição de caráter legal, tecnológico ou social. 
> 
> **OPEN KNOWLEDGE - 1996**

## 🎋Principais características

- O Acesso dos Dados deve ser Livre;
- O Formato dos Dados devem ser Abertos e Legíveis por Máquina; 
- Devem possuir Acesso Programático (como APIs);
- Devem possuir Contexto e Metadados; e
- Possuem Foco em Dados Brutos. 

## 🎋Exemplos

- **Portal Brasileiro de Dados Abertos**
	- Possuem Dados do Governo Federal, Estadual e Municipal
	- Temas como Educação, Habitação, Meio Ambiente, Saúde, entre outros.
	
- **IpeData - Instituto de Pesquisa Econômica Aplicada**
	- Ampla gama de Informações sobre Economia, Finanças, Desenvolvimento Regional e Social. 
	
- **MapBiomas**
	- Projeto colaborativo que usa imagens de satélite para Mapear o uso e cobertura da terra no Brasil de 1985. 
	
- **TerraBrasilis**
	- Plataforma interativa para acesso, consulta, análise e disseminação de dados da Vegetação Nativa do Brasil.
