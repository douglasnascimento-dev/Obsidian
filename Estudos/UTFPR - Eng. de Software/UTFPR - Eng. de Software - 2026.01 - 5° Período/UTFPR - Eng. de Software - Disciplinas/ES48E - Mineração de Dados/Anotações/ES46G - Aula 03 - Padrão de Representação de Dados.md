## 🌿Tipos de Dados

## 🎋 Estruturados

São dados organizados e armazenados em formato fixo predefinido, com uma estrutura descrita a partir de um modelo de dados. Como em uma Planilha ou em uma Tabela de um BD. São de fácil armazenagem, acesso e análise. Entretanto, possuem pouca flexibilidade para lidar com dados variáveis. 
## 🎋 Semiestruturados

A Organização dos dados é flexíveis, ou seja, não possuem um esquema rígido, mas também não são totalmente desestruturados. Como em XML's, JSON, Logs ou Emails. São mais flexíveis e fáceis de interpretar por máquinas e humanos. Entretanto, o processo é mais complexo que os dados estruturados. 
## 🎋 Não estruturados

São dados com formato livre e não seguem regras de organização. Como em PDF's e arquivos de textos, imagens, vídeos e áudios. Possuem alta flexibilidade. Entretanto, sua difícil indexação, acesso e análise, além de exigirem processamento de texto, visão computacional ou reconhecimento de voz, são algumas de suas limitações.
