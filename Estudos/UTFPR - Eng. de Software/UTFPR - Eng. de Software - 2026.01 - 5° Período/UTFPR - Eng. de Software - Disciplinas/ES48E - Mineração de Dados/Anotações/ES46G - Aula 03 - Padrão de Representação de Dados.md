## 🌿Tipos de Dados

## 🎋 Estruturados

São dados organizados e armazenados em formato fixo predefinido, com uma estrutura descrita a partir de um modelo de dados. Como em uma Planilha ou em uma Tabela de um BD. São de fácil armazenagem, acesso e análise. Entretanto, possuem pouca flexibilidade para lidar com dados variáveis.
