> A Mineração de Dados é um Passo no Processo de Descoberta de Conhecimento que consiste na realização de Análise dos Dados e na aplicação de Algoritimos de descoberta que, sob certas limitações computacionais, produzem um conjunto de Padrões de certas dados

## 🌿Processo KDD

KDD é um Processo não trivial, interativo e iterativo, para identificação de padrões compreensíveis, válidos, novos e potencialmente úteis a partir de grandes conjuntos de dados. A Partir de Dados Brutos, o KDD extrai informações que podem se transformar em conhecimento útil. 

1. **SELEÇÃO**
	- Nessa etapa é realizado a Identificação de Fontes e a Definição dos Objetivos.
	
2. **PRÉ-PROCESSAMENTO**
	- Nessa etapa é realizada a Limpeza de Dados (como a remoção de dados inconsistentes e tratamento de valores nulos), a Integração de Dados e a Redução de Dados (através da simplificação).
	
3. **TRANFORMAÇÃO**
	- Nessa etapa é realizada a Normalização dos Dados (como a padronização de escalas) e Seleção de Características.
	
4. **MINERAÇÃO**
	- Nessa etapa é realizada a Aplicação de Técnicas e Algoritimos (como os de Classificação para a Rotulação de Dados conforme categorias pré-definidas, ou por Agrupamento, para identificação de grupos de dados similares). 
	
5. **INTERPRETAÇÃO E AVALIAÇÃO**
	- Nessa etapa é determinada a Relevância dos Resultados, a Verificação dos Critérios e a Tradução dos Resultados
