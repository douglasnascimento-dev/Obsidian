## 🌿Listas

É uma Estrutura de dados que armazena uma sequência de itens (elementos). Cada item na lista fica em uma posição (índice), começando com o 0. E as Listas podem armazenar qualquer tipo de dado. 

## 🎋Operações Comuns em Listas

| Operação     | Descrição                                                                              |
| ------------ | -------------------------------------------------------------------------------------- |
| append(x)    | Adiciona *x* no final da lista                                                         |
| insert(i, x) | Insere *x *na posição *i*                                                              |
| remove(x)    | Remove a primeira ocorrência de *x*                                                    |
| pop(i)       | Remove e retorna o item na posição *i* (ou o último, se caso *i* não for especificado) |
| len(lista)   | Retorna o tamanho da lista                                                             |
| list\[i]     | Acessa o elemento no índice *i*                                                        |
