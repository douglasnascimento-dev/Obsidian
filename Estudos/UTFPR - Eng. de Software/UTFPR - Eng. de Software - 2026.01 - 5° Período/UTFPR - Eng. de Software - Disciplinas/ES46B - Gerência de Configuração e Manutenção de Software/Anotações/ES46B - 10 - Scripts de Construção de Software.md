## 🌿Scripts de Construção

São as instruções automatizadas que transformam o código-fonte em um pacote executável, rodando teste no meio do caminho. 

- **Maven** - Java
	- Muito estável e padronizado, mas os arquivos .xml podem se tornar extensos e verbosos. 

- **Grandle** - Java/Android/Kotlin
	- Combina a Flexibilidade de Scripts com a Gestão de Dependências do Maven. É muito mais rápido que o Maven, permite escrever lógica real no Build. 

- **WebPack** - JS/Front-end
	- É um empacotador de pacotes, suporta Atualização em Tempo Real e Carregamento sobre demanda. 

- **Make** - C++/Automações
	- Extremamente leve e altamente disponível em Unix

### 🎋**Compilação**

É a tradução. O Script pega o código legível para humanos e o transforma em código executável que o computador ou o servidor consegue entender
