# :LiAlbum: Itens de Configuração

Um Item de configuração é qualquer elemento de um sistema que precisa ser controlado e gerenciado durante o desenvolvimento e manutenção. Ou seja, são artefatos importantes para o sistema. Esses itens precisam de: Controle de Versão, Rastreamento de Mudanças e Identificação Única. São todos os Dados gerados durante o processo de desenvolvimento de software, e que se deseja rastrear. Podem ser divididos em 3 Principais Classes:
* Programas de PC
* Produtos que descrevem os programas (e infra); e
* Dados ou Conteúdo

## 🌿Processos

Envolve um conjunto de recursos e insumos usados para transformar algo em bens e serviços e é qualificado como um conjunto de atividades usadas para atingir uma determinada meta. 

Um Processo de Software pode ser definido como um conjunto estruturado de atividades necessárias para desenvolver, manter e evoluir um sistema de Software.
