## 🌳Diagrama de Atividades 

A Modelagem de atividades enfatiza a sequência e condições para coordenar comportamentos de baixo nível. Uma atividade é composta por um conjunto de ações, ou seja, os passos necessários para que a atividade seja concluída. O Objetivo do diagrama de atividade é mostrar o fluxo de atividades em único processo

- **Atividade** → especifica a coordenação de execuções de comportamentos subordinados usando um modelo de fluxo de controle e dados.
- **Nó de Ação** → são elementos mais básicos de uma atividade. Representa um passo de uma atividade
- **Fluxo de Controle** → o fluxo de controle é um conector que liga dois nós, enviando sinais de controle. É representado por uma linha de fluxo de controle contendo uma seta apontando para o nó alvo. 
- **Nó inicial** → ele representa o inicio do fluxo.
- **Nó final** → ele representa o fim do fluxo.
- **Nó de decisão** → também conhecido como nó de controle. É utilizado para representar um escolha entre dois ou mais fluxos possíveis. 
- **Nó de bifurcação/união** → é um nó de controle que pode dividir um fluxo em dois ou mais fluxos simultâneos e um nó de união mesclam dois ou mais fluxos concorrentes em um único fluxo de controle. 
- **Chamada de comportamento** → Uma ação de chamada de comportamento é uma ação que invoca a execução de um comportamento, sendo este, em geral, uma atividade. Representado por um ancinho. 
- **Participação de atividades** → permite presentar o fluxo de um processo que passar por diversos setores ou atores.
