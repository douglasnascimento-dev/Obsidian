## 🌳O Que é Requisitos?

São o ponto de partida para toda a definição do sistema. ==Um requisito é uma característica do sistema ou a descrição de algo que o sistema é capaz de realizar para atingir os seus objetivos.== Eles expressam as características e restrições do produto do ponto de vista da satisfação e necessidade do usuário

### 🌿Engenharia de Requisitos

É uma abordagem sistemática e disciplinada para a especificação e gerenciamento de requisitos. Busca conhecer os requisitos relevantes, estabelecer um consenso entre os [^1]Stakeholder a respeito de tais requisitos, documentar e gerenciar os requisitos.

[^1]*Stakeholders são pessoas, grupos ou entidades que têm interesse em uma empresa, projeto ou iniciativa*
