*Você é encarregado de desenvolver uma pequena aplicação para gerenciar o inventário de uma livraria. Para isso, você deve utilizar as seguintes técnicas do JavaScript: `.filter`, `.map`, `.reduce`, `Object.defineProperty` e uma função fábrica.*

## 🌹Instruções

1. **Crie uma Função Fábrica**  
    Desenvolva uma função chamada `Livro` (pode ser construtora ou fábrica) que cria objetos representando livros.  
    Cada livro deve ter as seguintes propriedades:
    
    - `titulo` (string): título do livro.
    - `autor` (string): autor do livro.
    - `genero` (string): gênero do livro.
    - `preco` (número): preço do livro.
    - `estoque` (número): quantidade em estoque.
    
    Utilize `Object.defineProperty` para:
    
    - Criar uma propriedade `disponivel` (boolean) que seja **calculada** automaticamente com base no valor de `estoque` (true se estoque > 0, false caso contrário).
    - Criar uma propriedade que não possa ser sobrescrita chamada `codigo` (string) que seja gerada automaticamente no formato `ISBN-XXXX` onde `XXXX` é um número aleatório de 4 dígitos.
	
2. **Filtrar Livros Disponíveis**  
    Crie uma função que use `.filter` para retornar uma lista com os livros que estão disponíveis no estoque.
    
3. **Mapear para Formatado Simples**  
    Crie uma função que use `.map` para transformar a lista de livros em um array de strings no seguinte formato:  
    `"Titulo: <titulo>, Autor: <autor>, Preço: R$<preco>"`.
    
4. **Calcular o Valor Total do Estoque**  
    Crie uma função que use `.reduce` para calcular o valor total do estoque, considerando todos os livros (quantidade em estoque multiplicada pelo preço).
    
5. **Simular Cadastro de Livros**  
    Escreva um código para simular o cadastro de pelo menos 5 livros no inventário e demonstre as funções de filtro, mapeamento e cálculo do estoque.
